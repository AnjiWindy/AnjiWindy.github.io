.hide()隐藏匹配的元素。 .show()显示匹配的元素 .toggle( [duration ][, complete ] )

如果没有参数，.hide()方法是最简单的方法来隐藏一个元素，.show()方法是最简单的方法来隐藏一个元素:

$('.target').hide();
$('.target').show();

如果没有参数，.toggle()方法是最简单的方法来切换一个元素可见性：

$('.target').toggle();

匹配的元素将被立即隐藏/显示，没有动画。这大致相当于调用.css('display', 'none')/.css('display', 'block')，但 display 属性值保存在 jQuery 的数据缓存中，所以 display 可以方便以后可以恢复到其初始值。如果一个元素的 display 属性值为 inline，那么隐藏再显示时，这个元素将再次显示 inline。

注意: 如果使用!important 在你的样式中，比如 display: none !important，如果你希望.show()方法才能正常工作，必须使用.css('display', 'block !important')重写样式。

当提供一个持续时间参数，.hide()/.show()就变成了一个动画方法。.hide()方法将为匹配元素的宽度，高度，以及不透明度，同时执行动画。一旦透明度达到 0，display 样式属性将被设置为 none，这个元素将不再在页面中影响布局。

持续时间是以毫秒为单位的，数值越大，动画越慢，不是越快。字符串 'fast' 和 'slow' 分别代表 200 和 600 毫秒的延时。

请注意，.hide()立即触发，并且如果没有 duration（持续时间）或 指定的 duration（持续时间）为 0，将覆盖动画队列。

如果提供回调函数，这个 callback 函数将在动画完成后被执行一次。这个能用来将不同的动画串联起来组成一个事件序列。这个回调函数不设置任何参数，但是 this 指向执行动画的 DOM 元素。如果多个元素一起做动画效果，值得注意的是这个回调函数在每个匹配元素上执行一次。这个动画不是作为一个整体。 我们可以给任何元素做动画，比如一个简单的图片：

<div id="clickme">
  Click here
</div>
<img id="book" src="book.png" alt="" width="100" height="123" />

$('#clickme').click(function() {
  $('#book').hide('slow', function() {
    alert('Animation complete.');
  });
});

$('#clickme').click(function() {
  $('#book').show('slow', function() {
    // Animation complete.
  });
});

$('#clickme').click(function() {
 $('#book').toggle('slow', function() {
 // Animation complete.
 });
});

点击链接隐藏/显示所有段落:

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>hide demo</title>
  <script src="https://code.jquery.com/jquery-1.9.1.js"></script>
</head>
<body>
  <p>Hello</p>
  <a href="#">Click to hide me too</a>
  <button id="btn1">Show it</button>
  <button id="btn2">Toggle</button>
  <p style="display:none">Here is another paragraph</p>
<script>
    $("p").hide();
    $("a").click(function ( event ) {
      event.preventDefault();
      $(this).hide();
    });

    $("btn1").click(function () {
      $("p").show("slow");
    });

    $("btn2").click(function () {
      $("p").toggle();
    });
</script>
</body>
</html>
