props

props 由父组件传递给子组件，其依旧属于父组件。子组件无法修改其内容。 可以通过 this.props 获取 props 中属性值。 自定义组件传递属性值：组件中通过 static propTypes 定义组件需要的属性，通过 static defaultProps 定义属性默认值

state

属于组件私有，其值可以改变。 在构造函数中初始化 state，在需要修改的时候调用 setState() 方法。setState 只会覆盖同名的属性。 state 用于存储可变属性，不可变的属性存储于 props 中。

ref

给组件定义 ref 属性。假设其值为 value。 通过 this.refs.value 获取组件的实例。
